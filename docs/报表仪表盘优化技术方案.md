## 一、问题分析

### 1.1 页面打不开（操作后无响应）

**根因定位**：

| 问题点 | 代码位置 | 原因分析 |
|--------|---------|---------|
| 仪表盘不存在时显示空白 | `Dashboard.tsx` L162-168 | `currentDashboard` 为 null 时只显示 Empty，没有引导用户操作 |
| API 错误未处理 | `useDashboard.ts` L76-85 | `getDashboard` catch 后返回 null，但没有错误状态反馈给用户 |
| 新建仪表盘后跳转白屏 | `DashboardList.tsx` L47-55 | `createDashboard` 后立即 `navigate`，但后端可能还未写库完成 |
| 编辑模式添加图表后卡死 | `Dashboard.tsx` L82-98 | `handleAddChart` 使用 `y: Infinity`，可能导致布局计算死循环 |

**关键代码问题**：

```tsx
// Dashboard.tsx - 问题1: 不存在时无引导
if (!currentDashboard) {
  return (
    <Empty description="仪表盘不存在" style={{ marginTop: 100 }}>
      <Button type="primary" onClick={() => navigate('/report/dashboards')}>
        返回列表
      </Button>
    </Empty>
  );
}

// Dashboard.tsx - 问题2: y: Infinity 可能导致布局死循环
const newChart: ChartConfig = {
  id: `chart-${Date.now()}`,
  type,
  title: `新图表-${type}`,
  datasetId: '',
  gridLayout: { x: 0, y: Infinity, w: 6, h: 4 },  // Infinity 可能导致布局计算异常
};
```

### 1.2 配置流程繁琐

**问题分析**：
- 添加图表需要手动输入 datasetId、xAxis、yAxis 等字段
- 没有数据预览功能，无法确认配置是否正确
- 筛选器和图表的联动逻辑不清晰

## 三、问题修复方案

### 3.1 页面打不开修复

**修复1：优化 Dashboard 加载状态**

```tsx
// Dashboard.tsx - 修复后
useEffect(() => {
  if (!id) {
    // 无 ID 时跳转到列表
    navigate('/report/dashboards');
    return;
  }
  
  setLoading(true);
  getDashboard(id)
    .then(dashboard => {
      if (!dashboard) {
        // 仪表盘不存在，不再卡在空白页
        message.warning('仪表盘不存在或已删除');
        navigate('/report/dashboards');
      } else {
        setFilters(dashboard.filters || []);
      }
    })
    .catch(error => {
      message.error('加载失败: ' + error.message);
      navigate('/report/dashboards');
    })
    .finally(() => setLoading(false));
    
  return () => setCurrentDashboard(null);
}, [id, getDashboard, navigate, setFilters, setCurrentDashboard]);
```

**修复2：解决布局 Infinity 问题**

```tsx
// Dashboard.tsx - handleAddChart 修复
const handleAddChart = useCallback((type: ChartConfig['type']) => {
  if (!currentDashboard) return;
  
  // 计算新图表位置：找到当前最大 y 值
  const maxY = currentDashboard.charts.reduce((max, chart) => {
    const chartBottom = (chart.gridLayout?.y || 0) + (chart.gridLayout?.h || 4);
    return Math.max(max, chartBottom);
  }, 0);
  
  const newChart: ChartConfig = {
    id: `chart-${Date.now()}`,
    type,
    title: `新图表-${type}`,
    datasetId: '',
    gridLayout: { x: 0, y: maxY, w: 6, h: 4 },  // 不再使用 Infinity
  };
  
  addChart(newChart);
  setAddingChart(false);
  chartForm.resetFields();
}, [currentDashboard, addChart, chartForm]);
```

**修复3：添加错误边界**

```tsx
// src/pages/report/components/DashboardErrorBoundary.tsx

export class DashboardErrorBoundary extends React.Component<
  { children: React.ReactNode; fallback?: React.ReactNode },
  { hasError: boolean; error?: Error }
> {
  constructor(props: any) {
    super(props);
    this.state = { hasError: false };
  }

  static getDerivedStateFromError(error: Error) {
    return { hasError: true, error };
  }

  render() {
    if (this.state.hasError) {
      return this.props.fallback || (
        <Result
          status="error"
          title="仪表盘加载失败"
          subTitle={this.state.error?.message}
          extra={<Button onClick={() => window.location.reload()}>刷新页面</Button>}
        />
      );
    }
    return this.props.children;
  }
}
```
